Tfrrs Cutoff and Scratch Prediction Algorithm

Last spring, I realized that all of my teammates were spending hours trying to manually deduce who would scratch (not compete) from the First Rounds of the NCAA Championships. I decided to develop an algorithm and code a program that automatically outputs the predicted scratches from each event and what the cutoff time will be. The algorithm ended up being almost perfect for deducing cutoff times! I plan to add even more functionality, like webscraping from each tfrrs profile to check for injuries, and an advanced user interface as we get closer to the Track Season. Stay tuned! Check out it's repoistory if you're interested!
